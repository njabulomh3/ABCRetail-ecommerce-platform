# ABC Retail - Azure E-Commerce Platform

## Project Overview
Cloud-based e-commerce platform built on Microsoft Azure. Demonstrates PaaS architecture, secure deployment, and scalable design for retail business.

## Technologies
- Frontend: ASP.NET Core MVC, C#, HTML/CSS
- Backend: C#, .NET 6.0
- Database: Azure SQL Database
- Cloud Services: Azure App Service, Blob Storage, Table Storage, Queue Storage
- Tools: Visual Studio, GitHub, Azure Portal

## Key Features
- User authentication and registration
- Product catalog with image upload
- Shopping cart with persistence
- Order processing with queues
- Admin dashboard
- Responsive design

## Getting Started

### Prerequisites
- Visual Studio 2022
- Azure account (free tier)
- .NET 6.0 SDK

### Installation
1. Clone repo: git clone https://github.com/njabulomh3/azure-ecommerce-platform
2. Open in Visual Studio
3. Update appsettings.json with your Azure credentials
4. Run: dotnet ef database update
5. Build and run

## Security Features
- HTTPS enforcement
- Input validation
- Role-based access control
- Azure SQL encryption
